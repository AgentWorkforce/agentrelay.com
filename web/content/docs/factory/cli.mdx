## Run

```bash
factory run-once            # one discover → triage → dispatch cycle
factory start --mode live   # the daemon
```

{/* Global flags: --config, --dry-run, --backend internal|relay */}

## Single issue

```bash
factory triage <KEY>
factory dispatch <KEY>
factory babysit <PR>
```

## Operate

```bash
factory status
factory loop / loop-status / kill-loop
factory reap-orphans
factory canary <KEY>
```
